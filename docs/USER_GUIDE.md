## 快速开始

### 第一步：启动网站

1. 打开项目文件夹
2. 打开命令行（终端）
3. 运行以下命令：
   ```
   npm run dev
   ```
4. 在浏览器中访问 `http://localhost:5173/`

### 第二步：添加新内容

你可以通过以下方式添加新的卡片：
- 使用自动化工具添加（推荐）
- 手动创建内容文件

## 添加新卡片

### 使用自动化工具（推荐）

这是最简单的方法，不需要了解任何技术细节。

#### 添加技术文档

1. 打开命令行（终端）
2. 运行以下命令：
   ```
   npm run add-card docs
   ```
3. 系统会自动创建一个新的技术文档卡片
4. 按照提示编辑卡片内容

#### 添加项目作品

1. 打开命令行（终端）
2. 运行以下命令：
   ```
   npm run add-card projects
   ```
3. 系统会自动创建一个新的项目作品卡片
4. 按照提示编辑卡片内容

#### 添加个人分享

1. 打开命令行（终端）
2. 运行以下命令：
   ```
   npm run add-card life
   ```
3. 系统会自动创建一个新的个人分享卡片
4. 按照提示编辑卡片内容

### 操作流程

1. **运行添加命令** - 选择你要添加的卡片类型
2. **系统创建卡片** - 自动生成包含默认信息的卡片
3. **编辑卡片内容** - 修改标题、描述、图片等信息
4. **生成内容** - 运行 `npm run generate-content`
5. **查看效果** - 在浏览器中刷新页面

## 编辑卡片内容

### 找到卡片文件

1. **技术文档**：在 `src/content/docs.ts` 文件中
2. **项目作品**：在 `src/content/projects.ts` 文件中
3. **个人分享**：在 `src/content/lifePosts.ts` 文件中

### 编辑内容

1. 用文本编辑器打开对应文件
2. 找到你刚添加的卡片（通常在文件末尾）
3. 修改以下信息：

#### 技术文档
- `title`：文档标题
- `summary`：简短描述
- `category`：分类（如：编程语言、操作系统）
- `tags`：标签（如：C++、教程）
- `level`：难度（beginner、intermediate、advanced）
- `markdown`：详细内容

#### 项目作品
- `title`：项目标题
- `summary`：项目描述
- `stack`：使用的技术
- `highlights`：项目亮点
- `gallery`：项目图片
- `links`：相关链接

#### 个人分享
- `title`：分享标题
- `date`：日期（格式：年-月-日）
- `tag`：标签（如：生活、摄影、户外）
- `summary`：分享内容
- `cover`：封面图片

## 添加图片

### 步骤

1. **准备图片** - 确保图片格式为 .png、.jpg 或 .jpeg
2. **复制图片** - 将图片复制到 `public` 文件夹
3. **使用图片** - 在卡片内容中使用 `/图片文件名` 作为图片地址

### 示例

- 图片文件：`public/my-photo.jpg`
- 在卡片中使用：`/my-photo.jpg`

## 生成网站内容

### 重要！每次修改内容后都要做这一步

1. 打开命令行（终端）
2. 运行以下命令：
   ```
   npm run generate-content
   ```
3. 等待几秒钟，看到成功信息后就完成了
4. 在浏览器中刷新页面查看更改

## 常见问题

### Q: 卡片添加后看不到？

A: 请确保运行了 `npm run generate-content` 命令，然后刷新浏览器。

### Q: 图片显示不出来？

A: 检查图片是否放在 `public` 文件夹中，图片地址是否正确（以 `/` 开头）。

### Q: 命令运行失败？

A: 检查命令是否正确输入，确保终端在项目根目录中运行。

### Q: 如何删除卡片？

A: 打开对应的数据文件，找到要删除的卡片，删除整个卡片对象，然后运行 `npm run generate-content`。

### Q: 如何修改已有卡片？

A: 打开对应的数据文件，找到要修改的卡片，直接编辑内容，然后运行 `npm run generate-content`。
