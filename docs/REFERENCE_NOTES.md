# 参考站点拆解笔记

## 可迁移模式

- 大图首屏 + 深色遮罩：保证视觉冲击和文字可读性并存。
- 卡片化信息架构：让文档、项目、生活分享在统一容器中表达。
- 动效分层：页面级过渡轻、局部交互细、彩蛋动效短促。

## 页面节奏借鉴

- 首屏只讲“我是谁”，次屏才进入“你能看什么”。
- 导航入口在次屏集中展示，降低用户认知负担。
- 彩蛋入口固定在边角位置，避免抢占内容焦点。

## 动效借鉴

- 进入动效采用位移 + 淡入，曲线统一。
- 列表筛选先淡出再淡入，避免内容瞬移。
- 彩蛋触发使用拖拽阈值，增强探索感。

## 不建议照搬

- 过多粒子背景或全局浮层效果
- 高频自动动画（影响阅读）
- 过强饱和度色彩大面积铺底
