## 设计理念

### 核心原则

1. **专业与亲和并重**
   - 主站表达"专业与可信"
   - 趣味层表达"亲和与记忆点"

2. **信息层次优先**
   - 先信息层次，再视觉装饰
   - 避免纯炫技动效

3. **双层结构设计**
   - 首页采用"首屏身份表达 + 下滑导航舞台"双层结构
   - 清晰的视觉层次

### 设计风格

- **现代简约**: 扁平化设计，减少视觉噪音
- **科技感**: 深色背景，科技蓝强调色
- **流畅交互**: 平滑的动画过渡
- **细节打磨**: 精致的微交互

## 色彩系统

### 主色调

```css
:root {
  /* 背景色 */
  --bg-0: #000000;        /* 主背景 */
  --bg-1: #050505;        /* 次要背景 */
  
  /* 表面色 */
  --surface: rgba(255, 255, 255, 0.08);      /* 表面 */
  --surface-1: rgba(255, 255, 255, 0.12);    /* 悬浮表面 */
  --surface-2: rgba(255, 255, 255, 0.16);    /* 激活表面 */
  
  /* 文本色 */
  --text: #ffffff;         /* 主文本 */
  --text-muted: rgba(255, 255, 255, 0.6);  /* 次要文本 */
  
  /* 强调色 */
  --accent: #0071e3;       /* 科技蓝（主操作） */
  --accent-hover: #0077ed; /* 悬浮状态 */
  
  /* 趣味色 */
  --play-accent: #ff6b6b;  /* 小游戏与彩蛋专用 */
  
  /* 线条色 */
  --line: rgba(255, 255, 255, 0.1);  /* 分割线 */
}
```

### 色彩使用原则

#### 主色（--accent）

**用途**：
- 导航激活状态
- 主操作按钮
- 链接高亮
- 重要信息强调

**示例**：
```css
.nav-pill.active {
  background: var(--accent);
}

.btn-primary {
  background: var(--accent);
}
```

#### 中性色（--bg-*、--surface*、--line）

**用途**：
- 背景层次
- 卡片表面
- 分割线
- 留白空间

**示例**：
```css
.card {
  background: var(--surface);
  border: 1px solid var(--line);
}
```

#### 趣味色（--play-accent）

**用途**：
- 小游戏元素
- 彩蛋入口
- 趣味交互

**注意**：
- 仅用于小游戏与彩蛋入口
- 不扩散到正文区
- 保持克制使用

## 排版规范

### 字体系统

```css
:root {
  /* 字体族 */
  --font-sans: -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif;
  --font-mono: "SF Mono", Monaco, "Cascadia Code", monospace;
  
  /* 字号 */
  --text-xs: 0.75rem;    /* 12px */
  --text-sm: 0.875rem;   /* 14px */
  --text-base: 1rem;     /* 16px */
  --text-lg: 1.125rem;   /* 18px */
  --text-xl: 1.25rem;    /* 20px */
  --text-2xl: 1.5rem;    /* 24px */
  --text-3xl: 1.875rem;  /* 30px */
  --text-4xl: 2.25rem;   /* 36px */
  
  /* 行高 */
  --leading-tight: 1.25;
  --leading-normal: 1.5;
  --leading-relaxed: 1.75;
}
```

### 标题层级

```css
h1 {
  font-size: var(--text-4xl);
  font-weight: 700;
  line-height: var(--leading-tight);
}

h2 {
  font-size: var(--text-3xl);
  font-weight: 600;
  line-height: var(--leading-tight);
}

h3 {
  font-size: var(--text-2xl);
  font-weight: 600;
  line-height: var(--leading-normal);
}

h4 {
  font-size: var(--text-xl);
  font-weight: 500;
  line-height: var(--leading-normal);
}
```

### 排版原则

1. **标题层级清晰**
   - 使用明确的字号差异
   - 保持视觉层次

2. **正文保持中等行高**
   - 降低阅读疲劳
   - 提高可读性

3. **卡片摘要保持短句**
   - 避免一屏信息过密
   - 突出重点信息

4. **标签胶囊统一视觉**
   - 减少模块割裂感
   - 保持一致性

## 动画规范

### 动画时长

```css
:root {
  /* 动画时长 */
  --duration-fast: 0.2s;
  --duration-normal: 0.3s;
  --duration-slow: 0.45s;
  --duration-slower: 0.6s;
}
```

### 标准时长

| 动画类型 | 时长范围 | 使用场景 |
|---------|---------|---------|
| 入场动画 | 0.45s ~ 0.6s | 页面进入、元素出现 |
| Hover 动画 | 0.2s ~ 0.28s | 鼠标悬浮、按钮交互 |
| 内容切换 | 0.2s ~ 0.24s | 筛选、标签切换 |
| 滚动动画 | 0.3s ~ 0.4s | 滚动出现、视差效果 |

### 缓动函数

```css
:root {
  /* 缓动函数 */
  --ease-out: cubic-bezier(0.33, 1, 0.68, 1);
  --ease-in-out: cubic-bezier(0.65, 0, 0.35, 1);
  --ease-bounce: cubic-bezier(0.34, 1.56, 0.64, 1);
}
```

**推荐缓动**：
- **power2.out**: 通用过渡
- **elastic.out**: 弹性效果
- **bounce.out**: 弹跳效果

### 动画工具库

使用 `src/shared/motion.ts` 统一管理动画：

```typescript
// 页面入场动画
animateViewEnter(container);

// 悬浮提升效果
bindHoverLift(selector);

// 内容切换动画
animateSwapOutIn(target, beforeIn);
```

### 动画约束

1. **单屏并行动画不超过 3 组**
   - 避免视觉混乱
   - 保持流畅性

2. **阅读型页面避免高频循环动画**
   - 不干扰阅读
   - 减少性能消耗

3. **宠物与小游戏动画可随时关闭**
   - 不影响主内容
   - 用户可控

4. **彩蛋入口保持克制**
   - "可发现但不打扰"
   - 默认固定在右下角纸角形态

## 响应式设计

### 断点系统

```css
/* 移动端 */
@media (max-width: 639px) {
  /* 小屏幕样式 */
}

/* 平板 */
@media (min-width: 640px) and (max-width: 1023px) {
  /* 中等屏幕样式 */
}

/* 桌面 */
@media (min-width: 1024px) {
  /* 大屏幕样式 */
}

/* 大屏 */
@media (min-width: 1280px) {
  /* 超大屏幕样式 */
}
```

### 响应式原则

1. **移动优先**
   - 先设计移动端
   - 逐步增强到桌面端

2. **弹性布局**
   - 使用 flexbox 和 grid
   - 避免固定宽度

3. **图片自适应**
   - 使用 max-width: 100%
   - 支持不同屏幕尺寸

4. **字体缩放**
   - 使用相对单位（rem、em）
   - 保持可读性

## 特殊元素设计

### 宠物系统

**设计原则**：
- 宠物是"嵌入式 UI 元素"
- 绑定在页面内容区域而非悬浮遮挡
- 可交互但不干扰主内容

**视觉风格**：
- 简约可爱的造型
- 柔和的颜色
- 流畅的动画

### 彩蛋入口

**设计原则**：
- 彩蛋是惊喜，不是主路径
- 入口视觉保持克制
- "可发现但不打扰"

**视觉风格**：
- 默认固定在右下角纸角形态
- 悬浮时有微妙提示
- 点击后展开完整入口

### 粒子系统

**设计原则**：
- 增强科技感
- 不干扰内容阅读
- 性能友好

**视觉风格**：
- 小圆点粒子
- 柔和的连线
- 鼠标交互效果
