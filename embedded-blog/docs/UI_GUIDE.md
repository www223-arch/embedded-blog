# UI 设计指南

## 设计理念

- 主站表达“专业与可信”，趣味层表达“亲和与记忆点”。
- 先信息层次，再视觉装饰；避免纯炫技动效。
- 首页采用“首屏身份表达 + 下滑导航舞台”双层结构。

## 色彩策略

- 主色：`--accent`（科技蓝）用于导航激活与主操作。
- 中性色：`--bg-*`、`--surface*`、`--line` 控制层次与留白。
- 趣味色：`--play-accent` 仅用于小游戏与彩蛋入口，不扩散到正文区。

## 排版策略

- 标题层级清晰，正文保持中等行高，降低阅读疲劳。
- 卡片摘要保持短句，避免一屏信息过密。
- 标签胶囊统一视觉，减少模块割裂感。

## 宠物与彩蛋原则

- 宠物是“嵌入式 UI 元素”，应绑定在页面内容区域而非悬浮遮挡。
- 彩蛋入口是惊喜，不是主路径，入口视觉保持克制。
